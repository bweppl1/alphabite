# AlphaBite

Entry for DonTheDeveloper Community Hackathon.

## Summary

AlphaBite is a gamified learning tool for Toddlers. It provides a flexible learning opportunity with progress incentives.
With repetition of short, phonetic words toddlers become more confident and interested in reading and spelling.

## MVP

### Spelling Game

- 10 Rounds; Player spells the emoji
- Receive points/coins based on performance

### Reading Game

- 10 Rounds; Player reads the three word options and chooses the word that represents the emoji
- Receive points/coiins based on performance

### Stats

Register to access:

- Track level in reading & spelling
- Track coins earned

## Future Features

1. Theme settings (animals, food, activities, colours)
2. Badge shop - Spend earned coins on badges
3. Parent word adjustments - Add or Remove words from the game list

## Future Enhancements

1. Track problem words
